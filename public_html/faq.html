What is YAMM?

YAMM is a free (both as in free beer and free speach) and open source money manager tool. It has been written by Benjamin Howe as part of his degree in Computer Science from the University of Reading.

How do I use YAMM?

Simply download the YAMM JAR file from the download page (and the Bouncy Castle JAR file, if you don't already have a copy), then click on the JAR file to run it. You will be prompted to select a directory for YAMM to save data in, and a password for YAMM to encrypt data with. The web interface will then be launched in your default browser. When you add new accounts, YAMM will "pop up" and ask for any details required.

Is YAMM secure?

We think so. Your security credentials only ever leave your computer to be sent to your financial institution - they are never sent to a third-party. When you quit YAMM, your security credentials are encrypted using the password you selected and saved in the folder you specified. We don't think YAMM is any more risky than "ordinary" online banking.

I forgot my password. Can you remind me what it is?

No! Your YAMM password is used to encrypt and decrypt the data files which YAMM creates. It is never sent to a third-party, nor can it be retrieved. You can reset your password by deleting all the files ending in .yamm from the YAMM data folder.

Which institutions does YAMM support?

At the moment, YAMM only supports Starling Bank. Support for other institutions is planned. To request support for a specific institution, contact us.

Starling: where do I find my personal access token?

Firstly, you need to create a Starling Developers account. Then, you need to connect your Starling Developers account to your Starling Bank account. Finally, you need to create a personal access token. YAMM requires the account:read, balance:read, and transaction:read scopes.
