Download

Please note: YAMM is currently in alpha testing. This means there may be severe issues with the code, and that the code has not been thoroughly checked for security issues. Proceed with caution!

In order to use YAMM, you need to install Java. We reccomend installing Java 9. If you prefer, you can install Java 8, but users of Oracle Java 8 will also need to install the Java Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files to allow YAMM to use encryption strong enough to protect your financial information. YAMM also works with OpenJDK. YAMM is not compatible with Java 7 or older as these versions of Java no longer recieve security updates.
