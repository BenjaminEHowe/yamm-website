YAMMBot

If you've landed on this page then you've probably noticed a strange user-agent in your log files. YAMMBot is the user-agent for the YAMM client application, which aggregates financial information on behalf of users. Security credentials (such as passwords and access tokens) are stored on the user's computer in AES-256 encrypted text files.

If you have more questions, or if you would like YAMM to stop accessing your websites, please contact us.

Created by Benjamin Howe. YAMM is open source software.
