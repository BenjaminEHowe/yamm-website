Open Source

YAMM is open source software, and licensed under the GNU General Public License version 3. This means you are entitled to view the source code behind YAMM, and can verify that it "does what it says on the tin". There are no secrets - if you do not trust the binaries available for download you can compile your own version of the client (from source code which can be audited).

YAMM's source code lives in two GitHub respositories:

  • BenjaminEHowe/yamm-client - the desktop application used to launch YAMM (and to handle stored data and communication with financial institutions).
  • BenjaminEHowe/yamm-website - this website, used to display general information about YAMM and to provide an interface to view the data obtained by the client.
